void requestCache(double latGridStart, double lonGridStart, double latGridEnd, double lonGridEnd)¶

Invoked when the model wants to cache certain region.

  • latGridStart – Start coordinate.

requestObjects¶

void requestObjects(double latGridStart, double lonGridStart, double latGridEnd, double lonGridEnd)¶

Invoked when the model wants to load objects for the given region. Objects should be set with MapTableModel addObjects method.

Parameters:
  • latGridStart – Start coordinate.
  • lonGridStart – Start coordinate.
  • latGridEnd – End coordinate.
  • lonGridEnd – End coordinate.
